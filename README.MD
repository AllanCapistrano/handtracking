# Hand Tracking

## :books: Descrição ##
Módulo para detecção de mãos utilizando [OpenCv](https://pypi.org/project/opencv-python/) e [Medipipe](https://mediapipe.dev/).

## Como utilizar

```powershell
pip install -U git+https://github.com/AllanCapistrano/hand-tracking.git
```
